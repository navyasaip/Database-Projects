Source code language: Java

Framework: Java Swings

OS: Windows

Software libraries: MySQL, Netbeans or Eclipse, mySql Java connector.

Steps to run:

1) Open mysql command line,
create the library database.

2) The GUI files are present in 'proj' folder.
Open NetBeans or Eclipse:
Import the source code and also have the mysql java connector library imported. 

In Sql_Connect.java change the url,user and password to setup the mySQL connection 

In the default package two java files are present:
--> Sql_Connect.java is used for connection to database to get the results.
--> NewJFrame.java is the main gui java file.

Run NewJFrame.java then a new window appears which is the java application of Library Management System.
